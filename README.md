# README

このリポジトリは、Kloud ハッカソン#3 で作成するサービスの API についての定義がまとめられています。

API に関する仕様はこのリポジトリを参照してください。

## API に付いて

| API 名 | メソッド | 機能                                   | レスポンス   |
| :----- | :------- | :------------------------------------- | :----------- |
| theme  | GET      | サーバーからテーマの一覧を取得します。 | テーマリスト |
| code   | POST     | サーバーに作成されたコードを送信します | スコア       |
